Tipos de dados

- O que são tipos de dados?

Os tipos de dados em C++ definem a natureza dos dados que podem ser armazenados em variáveis. Eles determinam o tamanho, o intervalo e as operações que podem ser realizadas sobre esses dados.

Dados Primitivos

Dados primitivos em C++ são tipos de dados básicos e pré-definidos pela linguagem, que representam valores simples. Estes tipos não são compostos e têm uma representação direta na memória.

  • Comentário de linha única: Usa // para comentar uma linha inteira. Tudo o que vem depois de // na mesma linha será ignorado pelo compilador.
  • Comentário de várias linhas: Usa /* para iniciar e */ para terminar o comentário, permitindo que o comentário ocupe várias linhas.

Comentário com //


// Este é um comentário de linha única
int x = 10;  // A variável x recebe o valor 10

Comentário com /* */


/*
Este é um comentário
de várias linhas.
Ele pode ocupar várias linhas sem afetar o código.
*/
int y = 20;

• Instalação do MinGW (Uma distribuição do GCC para Windows)

  • 1- Acesse o site do MinGW

    • Abra o seu navegador e vá para o site oficial do MinGW: MinGW.

  • 2- Baixe o instalador

    • Clique em "downloads" no menu e instale o instalador do MinGW (geralmente chamado mingw-get-setup.exe)

  • 3- Selecionar os pacotes

    • Durante a instalação, verá um ecrã de configuração. Selecione o "Basic Setup" no painel à esquerda

    No painel à direita, marque as caixas para os pacotes "mingw32-base" e "mingw32-gcc-g++"

    Clique com o botão direito do mouse sobre os pacotes selecionados e escolha "Mark for Installation

  • 4- Instale os pacotes

    • Após marcar os pacotes, clique em "Installation" no menu superior e selecione "Apply Changes".

  • 5- Adicionar o diretório ao PATH do sistema

    • - Clique com o botão direito no ícone do Windows e selecione "Sistema".

    - Clique em "Configurações avançadas do sistema" no painel à esquerda.

    - Na janela "Propriedades do Sistema", clique no botão "Variáveis de Ambiente".

    - Na seção "Variáveis do sistema", localize a variável chamada "Path", selecione-a e clique em "Editar".

    - Clique em "Novo" e adicione o caminho C:\MinGW\bin.

    - Clique em "OK" para fechar todas as janelas.

  • 6- Verificar a instalação

    • Abra o Prompt de Comando (cmd) e digite 'g++ --version' e Enter. Se a instalação foi bem-sucedida, verá a versão do GCC instalada

• Instalação do Visual Code

  • 1- Acesse o site do VS

    • Abra o seu navegador e vá para o site oficial do Visual Studio: Visual Studio

  • 2- Baixe o instalador

    • Clique no botão "Baixar" e localize o ficheiro instalado. Clique 2 vezes para iniciar a instalação

  • 3- Selecionar a carga de trabalho

    • Na tela de instalação, haverá várias opções de carga de trabalho. Selecione a opção "Desenvolvimento de Desktop com C++" e clique no botão "Instalar"
